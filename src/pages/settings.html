IT Staff Reports Access

IT Staff Import Access

Allow IT Staff to import assets and employees from CSV/Excel files

Permission Effects:

  • Reports Access Enabled: IT Staff can access Reports page, view all reports, and export data
  • Reports Access Disabled: Reports navigation hidden for IT Staff, access redirected to dashboard
  • Import Access Enabled: IT Staff can see and use Import buttons in Assets and Employees pages
  • Import Access Disabled: Import buttons hidden for IT Staff
  • Note: Admins always have full access regardless of these settings
  • Note: Viewers never have access regardless of these settings
